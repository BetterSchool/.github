# BetterSchool Project

![Cover](https://user-images.githubusercontent.com/62578958/178498117-79ab4b4e-a013-409c-ad67-b180907d614b.png)

Since the close of Xiaoheiban (晓黑板), schools need a new school-family contact application.
BetterSchool is such an application that can enable us to chat with our teacher / students.

The backend is built with [Axum](https://github.com/tokio-rs/axum) and [SeaORM](https://github.com/SeaQL/sea-orm).
And the frontend is built with native UI framework (Jetpack-Compose / SwiftUI).
Now we only decided to complete the mobile platform.
In the future version, we are going to complete the desktop platform and web.

We hope that we can take the place of Xiaoheiban and create a better channel for school and family.
